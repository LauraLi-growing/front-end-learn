  • margin

    （1）margin的不同取值个数分别代表的意思
    margin:10px
    margin:10px 20px
    margin:10px 20px 30px
    margin:10px 20px 30px 40px
    （2）盒子居中对齐和文字居中对齐
    盒子居中对齐：设置盒子定宽，再设置margin:0 auto；
    文字居中对齐：text-align:center
    （3）插入图片和背景图片的区别
    插入图片：通过img标签引入，大小使用width和height进行设置，位置使用margin进行设置，不会和文字产生重叠

    背景图片：通过background-url引入，大小使用background-size进行设置，位置使用background-position进行设置，遇到文字会在文字下方
    背景图片
    （4）清除内外边距：不同的网页展示时，会有不同的默认内外边距，要使页面显示统一，使用padding:0 0和margin:0 0来清除
    （5）行内元素上下内外边距的问题
    行内元素，尽量不要设置上下内外边距



    （6）外边距合并（塌陷）
    • 同级元素外边距合并：
      margin-top: 30px;
      margin-bottom: 20px;
      解决方法：只设置一边的边距
    • 父元素和子元素外边距合并：
      解决方法： 1、给父元素设置上边框(border)或者设置上内边距(padding) 2、给父元素设置overflow:hidden

计算盒子的外盒尺寸（空间尺寸）和内盒尺寸（元素大小）

  • 外盒尺寸：content+padding+border+margin
  • 内盒尺寸：content+padding+border
